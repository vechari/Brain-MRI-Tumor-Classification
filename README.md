# Brain-MRI-Tumor-Classification

The goal of this project is to train an image classification model to accurately detect a tumor in a brain MRI scan image. There will be two classification classes, “yes” and “no”. The aim is for a prediction accuracy over 95% to determine the validity of the model. 


Note: If the jupyter notebook file rapidly flickers, it is a issue with the level of zoom on your browser. If you zoom in or out the file should stabilize.
